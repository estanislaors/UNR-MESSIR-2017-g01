\subsection{Operation Model for init}

\label{OM-init}


The \msrcode{init} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{init}{
	used to initialize the current object as a new instance of the ctPoliceHeadquarter type. }

	\addrowheading{Parameters}
	\addnumbereddoublerow{}{Aid: dtPoliceHeadquarterID}{ a unique police headquarter identification} 
	\addnumbereddoublerow{}{Aname: dtString}{ complete name for the police headquarter} 
	\addnumbereddoublerow{}{Alocation: dtGPSLocation}{ Gps coordinates for police headquarter location} 
	\addnumbereddoublerow{}{AphoneNumber: dtPhoneNumber}{ police headquarter's telephone number for main communication} 

	\addrowheading{Return type}
	\addsinglerow{ptBoolean}

		

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{true iff the system poststate includes the current object as a new ctPoliceHeadquarter instance having its attributes equal to the ones provided as parameters. }

	\end{operationmodel}
